🎓 College Enquiry Chatbot (Tkinter)

A simple Python GUI chatbot that answers common college questions like courses, fees, hostel, library, and admission using Tkinter.

This project was created for B.Sc Computer Science Semester I.

📌 Project Description

This is a rule-based chatbot built using Python Tkinter.
It provides quick answers to frequently asked college enquiries through a simple chat interface.

The project is divided into multiple modules, where each file shows an improved version of the chatbot.

📂 Files in this Project

basicchatbot.py → Simple chatbot with dictionary responses

MultipleCategorieschatbot.py → Chatbot with multiple categories

scrolledtextchatbot.py → Scrollable chat window with colors

avatarchatbot.py → Chatbot with user & bot icons

🛠 Technologies Used

Python 3

Tkinter (GUI library)

🚀 How to Run

Install Python

Download the project

Open terminal in project folder

Run any file:

python basicchatbot.py


or

python avatarchatbot.py
